HOW TO RUN FRONTEND?
1. Open Terminal(MacOS) or Command Line(Windows), and enter into the Moodly folder.
2. cd client (Enter into directory)
3. RUN-npm i (installs all the packages) (if you don't have npm - first install that)
4. RUN-npm run start
THE SERVER WILL START RUNNING.
5. RUN-npm run build ( to build the project )


HOW TO RUN BACKEND?( to ensure that the login and register portion of the website runs)
1. Open Terminal(MacOS) or Command Line(Windows), and enter into the Moodly folder.
2. RUN-npm i (installs all the packages) (if you don't have npm - first install that)
3. RUN-npm run start
YOUR SERVER WILL START RUNNING.
